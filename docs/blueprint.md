# **App Name**: Ficha Matrimonial

## Core Features:

- Formulario de Matrimonio: Formulario unificado para ingresar datos del matrimonio, datos personales de él y ella, dirección, escalera de crecimiento, retiros como servidores, secretarías y áreas, grupos de crecimiento animados, y observaciones.
- Validación de Datos: Validar que los campos obligatorios no estén vacíos y que el año esté entre 1970 y 2035. Validar que no existan registros parciales en listas dinámicas. AI Tool which validates fields to ensure that there are no invalid or potentially incorrect data.
- Retiros como Servidores: Permite ingresar múltiples registros de retiros servidos, incluyendo año, mes, rol y comentarios.
- Secretarías y Áreas: Permite registrar la participación en secretarías y áreas, incluyendo el año y si participa en la asamblea.
- Grupos de Crecimiento: Registra los grupos de crecimiento animados, incluyendo el nombre del grupo, fecha de inicio, fecha de fin y encuentro.

## Style Guidelines:

- Primary color: Dark blue (#00308F) to reflect the Movement's logo and provide a sense of trust and stability.
- Secondary color: Yellow (#FFDA63) as an accent to highlight important information and interactive elements.
- Background color: Light gray (#F0F0F0) to ensure readability and reduce eye strain.
- Body font: 'Roboto' for clear and accessible text throughout the form.
- Headline font: 'Montserrat' for a modern and readable header style.
- Use icons that are consistent with the Movement's brand, focusing on simplicity and clarity.
- Divide the form into clear, distinct sections using visual separators and spacing to improve readability.
- Use subtle transitions and animations to provide feedback and guide the user through the form.
